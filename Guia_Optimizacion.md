# NEXO DYNAMIC | Guia Pro de Optimizacion de FPS y Fotogramas.

## 多Que es Nexo Dynamic FPS Booster?
Este conjunto de scripts ha sido dise単ado para maximizar el rendimiento de tu PC, eliminando procesos innecesarios de Windows, optimizando la comunicacion de red y priorizando el uso de la CPU para tus juegos favoritos.

## 多Que hace exactamente el script?

### 1. Plan de Energia "Nexo Ultra"
Activa el modo de rendimiento maximo oculto de Windows, asegurando que tu procesador no limite su potencia para ahorrar energia.

### 2. Optimizacion de Red (Ping Bajo)
Deshabilita el "Network Throttling", un proceso de Windows que limita la velocidad de red cuando el procesador esta bajo carga, asegurando un ping mas estable en juegos online.

### 3. Ajustes de Registro (CPU Priority)
Configura el sistema para dar prioridad absoluta a la ventana activa (tu juego) por encima de cualquier proceso en segundo plano.

### 4. Deshabilitacion de Telemetria y DVR
Windows envia constantemente datos de uso a Microsoft. Nuestro script detiene estos servicios para liberar ciclos de CPU. Ademas, deshabilita el Game DVR que suele causar tirones (stuttering) en graficas NVIDIA y AMD.

### 5. Limpieza Profunda
Elimina gigabytes de archivos basura temporales y cache antigua (Prefetch) que solo ralentizan el acceso a disco.
